Variables y Funciones Modernas en CSS

En esta página encontrarás TODO lo relacionado con variables CSS, la función var(), y las funciones matemáticas modernas calc() y clamp().

1. ¿Qué son las Variables CSS?

Las variables CSS, también llamadas *custom properties*, permiten almacenar valores reutilizables en tu hoja de estilos.

Se definen con prefijo --nombre y se usan con var().

Ejemplo básico:

:root {
    --color-primario: #3498db;
    --tamano-texto: 18px;
}

body {
    color: var(--color-primario);
    font-size: var(--tamano-texto);
}

Ventajas

  • Reutilización de valores.
  • Cambios globales rápidos.
  • Permiten temas claros/oscuro.
  • Admiten cálculos dinámicos con calc() y clamp().
  • Se pueden actualizar con JavaScript.

2. Uso de :root

:root es el selector del nivel más alto del documento y es el mejor lugar para definir variables globales.

Ejemplo:

:root {
    --fondo: #ffffff;
    --texto: #222;
    --espaciado: 20px;
}

3. La función var()

Sirve para acceder al valor de una variable CSS.

color: var(--texto);
margin: var(--espaciado);

Valor por defecto (fallback)

Si una variable no está definida, puedes usar un valor alternativo:

color: var(--color-secundario, red);

4. La función calc()

Permite realizar operaciones matemáticas directamente en CSS.

Operaciones soportadas:

  • Suma: +
  • Resta: -
  • Multiplicación: *
  • División: /

Ejemplo:

div {
    width: calc(100% - 40px);
    font-size: calc(var(--tamano-texto) * 1.2);
}

Notas importantes:

  • Debe haber espacios entre operadores (calc(100%-20px) ❌).
  • Se puede mezclar unidades (px, %, rem, etc.).

5. La función clamp()

Define un valor “fluido” con límites mínimo y máximo.

Formato:

clamp(valor_mínimo, valor_preferido, valor_máximo)

Ejemplo práctico (tipografía fluida):

h1 {
    font-size: clamp(24px, 5vw, 48px);
}

Esto significa:

  • Nunca será más pequeño que 24px.
  • Intentará ser igual al 5vw (valor fluido).
  • No crecerá más de 48px.

Ejemplo con variables:

:root {
    --min: 300px;
    --ideal: 50vw;
    --max: 900px;
}

section {
    width: clamp(var(--min), var(--ideal), var(--max));
}

Conclusión

Las variables y funciones modernas permiten un CSS más limpio, modular, escalable y fácil de mantener. Son esenciales para diseño responsive, temas, sistemas de diseño y layouts avanzados.
