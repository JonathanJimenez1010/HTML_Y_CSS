Guía Completa de CSS Grid

CSS Grid es uno de los sistemas de diseño más potentes y modernos para crear layouts bidimensionales. Permite controlar filas, columnas, áreas y alineación con gran facilidad.

1. display: grid

Para activar CSS Grid en un contenedor:

.container {
    display: grid;
}

Esto convierte todos los elementos internos en ítems de grid.

2. grid-template-columns y grid-template-rows

Definen el número y tamaño de columnas y filas.

Columnas

.container {
    display: grid;
    grid-template-columns: 200px 200px auto;
}

Los valores más usados:

  • px – tamaño fijo
  • % – relativo al contenedor
  • fr – fracción del espacio disponible
grid-template-columns: 1fr 1fr 1fr; /* 3 columnas iguales */

Filas

.container {
    grid-template-rows: 100px auto 50px;
}

3. Funciones avanzadas: repeat(), minmax(), auto-fit y auto-fill

a) repeat()

Sirve para repetir columnas o filas sin escribirlas una por una.

grid-template-columns: repeat(3, 1fr);

b) minmax()

Permite definir un rango mínimo y máximo.

grid-template-columns: repeat(3, minmax(150px, 1fr));

c) auto-fill y auto-fit

Se utilizan para crear grids responsivos automáticos.

auto-fill

Intenta meter la mayor cantidad de columnas posible aunque queden "huecos".

grid-template-columns: repeat(auto-fill, minmax(150px, 1fr));

auto-fit

Las columnas sobrantes colapsan para ajustarse al espacio disponible.

grid-template-columns: repeat(auto-fit, minmax(150px, 1fr));

4. Alineación en Grid

Grid permite alinear filas, columnas y contenido completo:

Alinear contenido dentro del grid:

.container {
    justify-content: center; /* eje horizontal */
    align-content: center;   /* eje vertical */
}

Alinear celdas internas:

.container {
    justify-items: center; /* horizontal */
    align-items: center;   /* vertical */
}

Alineación individual:

.item {
    justify-self: end;
    align-self: start;
}

5. Áreas en Grid

Permiten crear layouts complejos nombrando zonas del grid.

Definir áreas:

.container {
    display: grid;
    grid-template-columns: 1fr 2fr 1fr;
    grid-template-rows: auto auto;
    grid-template-areas:
        "header header header"
        "sidebar content ads";
}

Asignar cada área:

.header { grid-area: header; }
.sidebar { grid-area: sidebar; }
.content { grid-area: content; }
.ads { grid-area: ads; }

Esto produce un layout completo y estructurado fácilmente.

6. Otras propiedades importantes

  • gap – separaciones entre celdas
  • row-gap y column-gap
  • grid-auto-rows y grid-auto-columns
  • grid-auto-flow – controla cómo se colocan los ítems automáticamente
.container {
    gap: 20px;
    grid-auto-flow: dense; /* rellena huecos */
}

7. Ejemplo completo de Grid responsivo

.grid {
    display: grid;
    gap: 20px;
    grid-template-columns: repeat(auto-fit, minmax(200px, 1fr));
    padding: 20px;
}

Este layout crea tarjetas que se ajustan automáticamente al tamaño disponible.

Conclusión

CSS Grid es una herramienta increíblemente poderosa para crear layouts flexibles, ordenados y responsivos. Con estas propiedades y funciones puedes construir páginas profesionales de manera sencilla y limpia.
