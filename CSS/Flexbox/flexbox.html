Guía Completa de Flexbox

Explicación detallada, ejemplos en vivo y snippets para: display:flex, ejes, justify-content, align-items, flex-direction, flex-wrap, gap, flex-grow, flex-shrink y flex-basis.

1) ¿Qué es Flexbox y por qué usarlo?

Flexbox (Flexible Box Layout) es un modelo de distribución de elementos en 1 dimensión (fila o columna). Está pensado para gestionar alineación, distribución del espacio y orden de elementos dentro de un contenedor, incluso cuando la pantalla cambia de tamaño.

  • Perfecto para componentes, barras de herramientas, menús y layouts internos.
  • Trabaja en un solo eje a la vez: eje principal y eje cruzado.

2) Cómo activar Flexbox

Convierte un contenedor en flex contenedor con display: flex; o display: inline-flex;.

<div class="contenedor">
  <div class="item">1</div>
  <div class="item">2</div>
  <div class="item">3</div>
</div>

.contenedor{
  display:flex; /* o inline-flex */
}

3) Ejes: eje principal y eje cruzado

Eje principal (main axis) es la dirección en la que fluyen los items (por defecto row). Eje cruzado (cross axis) es perpendicular al principal.

eje principal — dirección del flujo;
eje cruzado — alineación perpendicular.

4) flex-direction

Controla la dirección del eje principal.

flex-direction: row | row-reverse | column | column-reverse;
  • row — izquierda a derecha (por defecto).
  • row-reverse — derecha a izquierda.
  • column — arriba a abajo (eje principal vertical).
  • column-reverse — abajo a arriba.

5) flex-wrap

Por defecto los items no se rompen de línea (nowrap). flex-wrap permite que los items pasen a la siguiente línea.

flex-wrap: nowrap | wrap | wrap-reverse;
  • nowrap — todos en una sola línea (default).
  • wrap — permite que los items se envuelvan en varias filas/columnas.
  • wrap-reverse — envuelto en sentido inverso.

6) justify-content (distribución en el eje principal)

Controla cómo se distribuye el espacio a lo largo del eje principal.

justify-content: flex-start | flex-end | center | space-between | space-around | space-evenly;
  • flex-start — items al inicio.
  • flex-end — items al final.
  • center — centrados.
  • space-between — espacio igual entre items (extremos pegados a contenedor).
  • space-around — espacio alrededor (mitad en los extremos).
  • space-evenly — espacios iguales tanto entre items como en los bordes.

7) align-items (alineación en el eje cruzado)

Alinea los items en el eje cruzado (perpendicular al eje principal).

align-items: stretch | flex-start | flex-end | center | baseline;
  • stretch — los items estiran para llenar (por defecto si no tienen altura fija).
  • flex-start — al inicio del eje cruzado.
  • flex-end — al final del eje cruzado.
  • center — centrados en el eje cruzado.
  • baseline — alineación por línea base del texto.

Nota: para alinear individualmente un item usa align-self.

8) gap (espaciado entre items)

Define separación entre los items sin usar márgenes individuales. Funciona en flex desde 2020+ en navegadores modernos.

gap: 12px; /* también row-gap, column-gap */

9) flex: shorthand para grow, shrink y basis

flex es la forma abreviada de: flex-grow flex-shrink flex-basis.

/* Ejemplos */
.item{ flex: 1; }         /* 1 1 0%  -> crecerá por igual */
.item{ flex: 2; }         /* 2 1 0%  -> crecerá el doble que flex:1 */
.item{ flex: 0 0 200px; } /* no crecer, no encoger, base 200px */

a) flex-grow

Controla cuánto espacio extra obtiene un item cuando hay espacio libre. Valor numérico (0 por defecto).

b) flex-shrink

Controla cuánto encoge un item cuando falta espacio. Valor numérico (1 por defecto).

c) flex-basis

Tamaño inicial del item antes de distribuir espacio libre. Puede ser auto, px, %. También suele usarse con flex shorthand.

Regla práctica: usa flex: 1 para items que deben compartir espacio igualmente; usa flex: 0 0 auto para items con tamaño fijo.

10) Propiedades útiles relacionadas

  • order: cambia el orden visual de los items (sin alterar DOM).
  • align-self: sobreescribe align-items para un item.
  • align-content: afecta el empaquetamiento en el eje cruzado cuando hay varias líneas (solo cuando hay wrap y espacio extra en el eje cruzado).

12) Errores comunes y soluciones

  • No funciona? Asegúrate de que display:flex está en el contenedor, no en los items.
  • Si align-items:stretch no estira, el item puede tener una altura/anchura explícita.
  • Usar gap evita márgenes colapsados entre items y es más limpio que aplicar margen a cada item.
  • Recuerda que justify-content actúa en el eje principal y align-items en el eje cruzado.

13) Consejos de rendimiento y accesibilidad

  • Flexbox es rápido y nativo: evita over-optimizing con scripts para layout.
  • No uses order si dependes del orden lógico para lectores de pantalla; altera sólo presentacionalmente.
  • Prueba con lector de pantalla y navegación por teclado para componentes flexibles (focus visibles y orden lógico).

14) Atajos y patrones comunes

  • .row-center { display:flex; align-items:center; justify-content:center; }
  • .space-between { display:flex; justify-content:space-between; align-items:center; }
  • Botones en barra: gap + align-items:center para espaciado consistente.
